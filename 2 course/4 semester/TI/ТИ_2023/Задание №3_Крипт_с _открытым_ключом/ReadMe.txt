Тема: Криптографические системы с открытым ключом.
Изучить теоритеческий материал для выполнения задания №3 (файл практикум_криптография.pdf, стр.14-20).

Выполнить задание согласно выданному варианту. 

Для успешной защиты необходимо иметь:
- корректно работающую программу, в которой учтены все требования согласно заданию;
- тесты и файлы для тестирования программы;
- знание теоретического материала по теме работы (знать и уметь выводить формулы по всем трем алгоритмам);
- примеры, иллюстрирующие работу следующих алгоритмов: быстрое возведение в степень, расширенный алгоритм Евклида, алгоритм поиска первообразных корней по модулю;
- при использовании длинной арифметики подготовить тестовые примеры для длинных p  и q.
